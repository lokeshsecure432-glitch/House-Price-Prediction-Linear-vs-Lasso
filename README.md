#  House Price Prediction

##  Overview
This project predicts house prices using Linear and Lasso Regression models.

##  Models Used
- Linear Regression  
- Lasso Regression (with LassoCV for automatic alpha selection)

##  Key Insight
Lasso eliminated 11 out of 34 features, yet both models achieved similar Adjusted R² scores.

##  Conclusion
Feature reduction did not improve performance, indicating that most features were already relevant.  
This highlights that simpler models can perform equally well on clean datasets.
